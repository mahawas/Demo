Web 2.0 Touch

Web 2.0 Touch is a concise mini JavaScript framework for touch devices (iPhone, iPad, Android). It is an open source project and you can participate in its development and/or testing.

https://github.com/web20boom/Web-2.0-Touch .

This particular build was done with the help of PhoneGap that makes it possible to distribute applications written with Web 2.0 Touch in Apple Store.
